## ADDED Requirements

### Requirement: 列表卡斜杠命令
系统 SHALL 提供一个斜杠命令“列表卡”，用于将当前块转换为 `#card(type=list)` 列表卡，并初始化列表条目的 SRS 状态。

#### Scenario: 通过斜杠命令创建列表卡
- **GIVEN** 用户光标位于一个块
- **WHEN** 用户执行斜杠命令“列表卡”
- **THEN** 系统为该块添加 `#card` 标签
- **AND** 系统将 `#card` 的 `type` 属性设置为 `list`
- **AND** 若存在条目子块，系统初始化条目 1..N 的 SRS 状态
- **AND** 系统向用户展示成功提示（无条目时提示用户后续添加子块作为条目）

### Requirement: 条件不满足时的提示
系统 SHALL 在当前块缺少条目时仍允许创建列表卡，并提供可操作的提示引导用户后续添加条目子块。

#### Scenario: 无条目时创建成功并提示
- **GIVEN** 用户光标位于一个块
- **WHEN** 用户执行斜杠命令“列表卡”
- **THEN** 系统应创建列表卡
- **AND** 系统应提示用户“请在该块下添加子块作为条目”
